# Consultor de Processos

**Versão:** 0.8.0
**Status:** Em planejamento

Sistema para monitoramento automático de processos judiciais em diferentes tribunais brasileiros.

## Objetivos

- Arquitetura modular
- Alta escalabilidade
- Crawler desacoplado
- Providers independentes
- Backend Java
- Frontend Ionic
